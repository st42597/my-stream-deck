# SLACK ACTIVITY

Reads the Slack Dock badge directly — so the count matches your Slack notification preferences exactly (mutes, keywords, DnD, channel settings).

Required: Accessibility permission

  1. System Settings → Privacy & Security → Accessibility
  2. Enable Elgato Stream Deck (and allow any child process prompts)
  3. Slack app must be running (badge appears on Dock icon)
First press may trigger a permission prompt. Grant it, then the badge number appears within a few seconds.

Button action

Press the key to open Slack and jump to the next unread channel/DM (⌥⇧↓) in sidebar order — respects your mute and notification settings. Press repeatedly to cycle through unreads.
